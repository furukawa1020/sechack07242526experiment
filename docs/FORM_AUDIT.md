# Googleフォーム公開内容の参考確認

対象URL: `https://forms.gle/BeShY7cY5zMjunto9`

この文書はアプリ外で使うGoogleフォームの参考確認項目であり、formal productionのrelease/startゲート、倫理承認証跡、アプリによる承認判定ではない。

## 現行の正

ユーザーが2026-07-23に提示した研究説明を正とし、明示されたタイポを次のように修正する。

- 「3種類の提示」ではなく「4種類の提示」
- 第4条件として「クラウドで処理し、状態名を付けず、画面上のフグのふくらみで示す設定」を含める
- 各提示の直後に、今見た提示について11項目へ回答する
- 研究説明、参加同意、アンケートを同じGoogleフォームで扱う
- 年齢区分は任意

正規化済みの全文は[FORM_COPY_CORRECTED.md](FORM_COPY_CORRECTED.md)を参照する。

## アプリとの境界

- production設定は`formUrl=""`、`formAudit`なし
- 参加者UIとOperator UIにフォーム名、URL、QRコード、回答内容、回答状況を表示しない
- アプリはフォームをfetch、解析、監査、送信、完了確認しない
- フォーム内容のSHA-256、確認者情報、確認日、証跡保存先を設定、Git、CI、manifest、実験ログへ保存しない
- フォームの内容や到達性をformal productionの技術開始条件にしない

## アプリ外での確認項目

- [ ] 研究課題名、研究責任者、共同研究者、実施場所、日程、所要時間が研究責任者の管理する現行情報と一致
- [ ] 条件数が4で統一されている
- [ ] クラウド×画面上フグを含む4条件が説明されている
- [ ] 各提示直後に11項目へ回答する説明で統一されている
- [ ] 同意しない場合は実験を開始しない
- [ ] 年齢区分は任意
- [ ] 氏名、メール、住所、電話番号、学籍番号を収集しない
- [ ] Google Formsのメール収集を使用しない
- [ ] 心拍、体温、顔画像、動画、音声、生体データを取得しない
- [ ] 固定模擬データと比較用クラウド表示を説明している
- [ ] 実験アプリは固定模擬身体データを外部送信・保存しないことと、Googleフォーム回答はGoogleへ送信・保存されることを区別している
- [ ] 参加者向けに内部条件コードA/B/C/Dを表示しない
- [ ] 回答したくない評価項目を空欄にできる
- [ ] 中止、撤回、問い合わせ方法を説明している

この確認結果をアプリへ入力しない。承認資料と実施条件は本システム外で管理し、本アプリは倫理承認を検証したとは表示しない。
