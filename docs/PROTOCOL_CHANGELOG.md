# プロトコル変更履歴

## R8-010-2x2-mock-v2 — 2026-07-20

- 処理場所を一目で識別できるよう、クラウド条件へcloud icon、ローカル条件へdevice iconを追加。
- 両アイコンは単色線画とし、色、大きさ、線幅、占有枠、配置を同一に固定。
- 処理場所の値を保存・閲覧範囲より大きくしたが、cloud/local間の文字サイズ、ウェイト、余白は同一。
- 参加者向け固定文言、条件対応、提示順、タイミング、固定値、フグ動作、外部通信方針は変更していない。

参加者が見る刺激の顕著性が変わるため、実験の再現性を優先してプロトコルバージョンを更新した。

## R8-010-2x2-mock-v1 — 2026-07-19

- 固定模擬データ方式の初版を実装。
- 条件対応を A=cloud+label、B=local+label、C=local+puffer、D=cloud+puffer に固定。
- 提示順を ABDC / BCAD / CDBA / DACB に固定。
- 固定状態を 72 / 高ストレス / フグ正規化レベル0.60に設定。
- クラウド条件を比較用シナリオとし、実際の外部送信を行わない。
- MockPufferDeviceとUSBシリアルアダプタ境界を定義。

この版では実際の心拍・生体センサを使用しない。実センサ連携は別プロトコル版として審査・実装する。
